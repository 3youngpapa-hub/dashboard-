📡 화면공유 방법
📌 방법 A — GitHub Pages (인터넷 가능 시)
1단계: github.com 접속 → 무료 가입
2단계: 새 저장소(Repository) 생성 → 이름: dashboard
3단계: gimpo-v7.0.html 업로드 → 파일명을 index.html로 변경
4단계: Settings → Pages → Branch: main → Save
5단계: URL 생성: https://[사용자명].github.io/dashboard/
공유: 이 URL을 패드/노트북 브라우저에서 열면 상황판 보기 가능
⚠️ 실시간 동기화 아님 — 입력 내용은 노트북에서만 변경됨
📌 방법 C — HDMI 직접 연결 (현장 기본)
1단계: HDMI 케이블로 노트북 → TV/모니터 연결
2단계: 키보드 윈도우키 + P 누르기
3단계: 복제 선택 → 노트북과 TV 동일 화면
4단계: 상황판 실행 → F11로 전체화면
✅ 완전 실시간 — 가장 안정적, 인터넷 불필요
💡 권장 병행 운용:
현장 출동 시 → HDMI로 통제단 TV 연결 (실시간)
상황실/관계자 → GitHub URL로 원격 확인
